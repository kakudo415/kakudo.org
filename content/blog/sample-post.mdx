# サンプルMDXブログ記事

これはNext.jsとMDXを使用したサンプルブログ記事です。このファイルは、MDXの機能をテストするために作成されました。

## MDXの特徴

MDXはMarkdownの拡張版で、JSXを直接記述できます。これにより、Markdownの簡潔さとReactコンポーネントの柔軟性を組み合わせることができます。

### コードブロック

```javascript
// JavaScriptのコードブロック
function hello() {
  console.log('Hello, MDX!');
}

hello();
```

### リスト

以下は箇条書きリストの例です：

- 項目1
- 項目2
- 項目3
  - ネストされた項目1
  - ネストされた項目2

番号付きリストも作成できます：

1. 最初の項目
2. 2番目の項目
3. 3番目の項目

### 引用

> これは引用ブロックです。
> 複数行にまたがることもできます。

### テーブル

| 名前 | 説明 |
|------|------|
| MDX | JSXを含むMarkdown |
| Next.js | Reactフレームワーク |
| React | UIライブラリ |

## まとめ

MDXは、ブログ記事やドキュメントを作成するための強力なツールです。Markdownの簡潔さとReactの柔軟性を組み合わせることで、豊かなコンテンツを作成できます。
